# Security Policy

## Supported Versions

| Version | Supported |
| --- | --- |
| 0.5.x | Yes |
| Earlier versions | No |

## Reporting a Vulnerability

请使用仓库 **Security** 页面中的 **Report a vulnerability** 私下报告安全问题。不要创建公开 Issue，也不要在报告中附带真实密码、私钥、完整日志或可访问的服务器信息。

报告应尽量包含：

- 受影响版本和操作系统
- 最小复现步骤
- 可能的影响
- 建议修复方案（如有）

维护者会尽量在 7 天内确认报告，并在修复可用后协调公开披露。未经协调请不要发布可直接利用的细节。
