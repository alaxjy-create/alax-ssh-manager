# Code of Conduct

## Our Standards

参与者应保持尊重、专业和建设性。欢迎不同经验、背景和观点，不接受骚扰、歧视、威胁、人身攻击、公开他人隐私信息或蓄意破坏协作的行为。

## Project Discussions

- 针对代码和行为讨论，不针对个人。
- 提供可复现信息，尊重安全披露流程。
- 不在 Issue、PR 或日志中发布真实凭据和个人服务器资料。
- 维护者可以编辑、隐藏或删除违反准则的内容，并限制持续违规者参与项目。

## Enforcement

需要私下报告行为问题时，请通过仓库 Security 页面创建私人报告。所有报告会在合理范围内保密处理。

本准则适用于项目仓库、Issue、Pull Request、Discussion 和代表项目参与的其他公共空间。
